---

When modifying the Tinode Web client:
- Prefer small UI changes.
- Preserve backend API and protocol compatibility.
- Keep server URLs configurable through environment/config files.
- Preserve Russian localization support.
- Avoid adding new UI frameworks unless the user asks.
- Inspect package.json and lockfiles before choosing npm/pnpm/yarn.
- Run available build/lint commands when practical.
- Report changed files and screenshots/manual test notes when available.
